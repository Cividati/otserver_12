# Forgotten Server
Servidor de tibia 

## Características
- Tibia 12
- Map Global
- Gesior ACC
- Site próprio

## Como usar?
- Baixe o repositório como zip;
- Extraia o ```otserver_12.zip``` em uma pasta;
- Extraia o arquivo ```data/world/world.zip```;
- Configure o arquivo ```config.lua```;
- Have fun! ^^
